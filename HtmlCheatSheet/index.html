HTML Tags Cheat Sheet

Structure

Tag Name Description
<!DOCTYPE html> Document Type Tells the browser to expect HTML document. Should be at the top.
<html> HTML The root of an HTML document. Should contain all other elements except DOCTYPE.
<head> Head Contains metadata about the document.
<body> Body Contains the displayed part of the document. There can only be one.
<div> Div Short for division, a container that divided document into sections.
<!-- --> Comment Contains a comment, not rendered in browser.

Metadata

Tag Name Description
<title> Title The title shown in the browser's title bar or page's tab.
<link> Link Links to another document such as a stylesheet.
lang=""; Language An attribute of the <html> element noting the language. "en" for English.

Styling and Selectors

Tag Name Description
<style> Style Contains CSS styling for all or part of a document.
style="" Inline Style Attribute containing CSS styling directly inside an element.
id="" ID Specific identifier for one element, must be unique in document.
class="" Class Identifier that can be used on multiple elements to creat a group.
<span> Span Selects a group of text for styling.

Semantic HTML

Tag Name Description
<header> Header Contains header. May contain title, logo, and navigation.
<nav> Navigation Contains navigation links for site or page.
<main> Main Contains the main content of the page.
<footer> Footer Contains footer. Usually author info, copyright data, or links.
<figure> Figure Used to encapsulate media.
<figcaption> Figure Caption Normally used inside <figure> to pair a caption with a media element.
<section> Section Used to divide document by theme such as an articles section and comments section.
<article> Article More specific than a section, might contain an individual article or comment.
<asside> Asside Contains additional material not essential understanding the main content.

Text Content

Tag Name Description
<p> Paragraph Usually contains block of text but can also contain other media.
<h1>...<h6> Headings Headings, <h1> being the largest and <h6> the smallest.
<strong> Strong Indicates important content. Browsers typically render in bold type.
<em> Emphasis Marks text as emphasized. Browsers typically render in italics.
<br> Line Break Jumps to next line in text.
<a> Anchor Creates a hyperlink to another page or location on same page. Can also be used on images.
href="" Hyperlink Reference Attribute of the <a> element, specifying where the link should go.
target="" Target Attribute of the <a> element, specifying where the hyperlink should be opened. "blank_" for new tab.

Media Content

Tag Name Description
<img> Image Embeds an image into the document. Requires src attribute and doesn't use closing tag.
<video> Video Embeds media player for video content. Use src to point to source.
<audio> Audio Embeds audio content. Use src to point to source.
<embed> Embed Can be used to embed any kind of media.
src="" Source Attribute that sets the source url for media content.
alt="" Source Attribute of <img>. Displayed if image doesn't load, also used in screen readers.
controls Conrols Attribute for video and audio which adds playback controls.

Tables

Tag Name Description
<table> Table Creates a table in document.
<thead> Table Head Contains the rows defining the column headers.
<tbody> Table Body Contains the rows with the table data.
<tr> Table Row Contains a row of the table.
<td> Table Data Contains a data element in the table.
<tfoot> Table Foot Contains the rows of the table footer.

Lists

Tag Name Description
<ul> Unordered List Typically rendered as bulleted list.
<ol> Ordered List Typically rendered as a numbered list.
<li> List Item Represents an item in a list.
